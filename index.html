Q8. Whatâ€™s the difference between window vs document??

Ans. -

Window

  1. Represents the global window in a web browser.
  2. Serves as the top-level object in the browser's window hierarchy.
  3. Contains properties and methods related to the browser window, such as dimensions, location, history, and navigation.
  4. Provides access to other global objects, such as document, navigator, and localStorage.
  5. Acts as the execution context for global JavaScript code.
  6. Handles events related to the window, such as load, resize, and scroll.
  7. Can be used to open new browser windows or tabs using methods like open() or close().

Document

  1. Represents the web page loaded within the browser window.
  2. Provides access to the elements and content within the web page.
  3. Contains methods and properties to manipulate and interact with the document's structure, content, and styling.
  4. Enables the selection of HTML elements using various methods like getElementById(), querySelector(), or getElementsByTagName()
  5. Allows modification of element properties, such as text content, HTML markup, styles, and attributes.
  6. Handles events that occur within the document, such as click, submit, or keydown.
  7. Provides methods to create new elements, modify the document structure, or dynamically load external resources.

Q9. Write code to implement timer clock using JS -- display HH:MM:SS -- the time should keep updating every second?

Ans. - 

                    function updateTime(){
                        var d = new Date();
                        var h = document.getElementById("hour");
                        var m = document.getElementById("minute");
                        var s = document.getElementById("sec");
                        h.textContent = d.getHours();
                        m.textContent = d.getMinutes();
                        s.textContent = d.getSeconds();
                      }
                      setInterval(updateTime, 1000);
